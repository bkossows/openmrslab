============================
OpenMRSLab 0.4 documentation
============================

OpenMRSLab brings together a collection of open source tools to support working
with Magnetic Resonance Spectroscopy data, including pre-processing raw data,
fitting and quantification, and generating output tables and images.

Getting help
============

Having trouble? We are here to help!

* Ask or search questions on the `MRSHub forum`_.
* Looking for information on data processing with Suspect? Try asking on the `suspect-users mailing list`_.
* Report bugs or request new features in our `issue tracker`_.

.. _MRSHub forum: https://forum.mrshub.org
.. _suspect-users mailing list: https://groups.google.com/forum/#!forum/suspect-users
.. _issue tracker: https://github.com/openmrslab/openmrslab/issues
